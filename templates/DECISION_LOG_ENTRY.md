**Rationale:** [Why this decision was made. Reference evidence, data, or precedent.]

- [File reference, data point, or lesson number]
- [File reference, data point, or lesson number]

**Impact:** [What changes downstream. Which chapters, which characters, which future books are affected?]

---

## Example

```
DEC-20260715-001: Adopt 9-Stage Pipeline as Default
**Decision:** The 4-pass editorial system expanded to 9 stages with Line Edit, Copy Edit, and Final QA added.
**Rationale:** Book 1's prose-level issues accumulated because there was no dedicated prose review stage. Book 2's structural fixes were sometimes applied during Line Edit, which blurred stage boundaries. The 9-stage system separates concerns.
**Evidence:**
- Book 1 editorial report: "prose rhythm issues not caught"
- Book 2 Line Edit report: "structural changes attempted during prose review"
- STORYCRAFT_HANDBOOK.md §Middle Act Sag
**Impact:** All future books use the 9-stage system. Book 3 was the first to use it fully.
**Status:** ACTIVE
**Recorded by:** Showrunner (Hermes)
**Date:** 2026-07-15
```
