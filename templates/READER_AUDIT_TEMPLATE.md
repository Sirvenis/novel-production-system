## Classification Table (For Showrunner)

The following table is completed by the showrunner/author, NOT the reader:

| # | Reader Finding | Showrunner Classification | Rationale |
|---|---------------|--------------------------|-----------|
| 1 | [Finding] | MUST FIX / INVESTIGATE / SHOULD / COULD / NOT A FIX | [Why] |

**Classification definitions:**
- **MUST FIX:** Implement immediately. Blocks freeze.
- **INVESTIGATE FIRST:** Try least invasive fix first. Escalate only if needed.
- **SHOULD FIX:** Individual judgment. Fix if time/capacity.
- **COULD FIX:** Polish. Defer if needed.
- **NOT A FIX:** Decline. The finding is noted but not acted upon.
