## The Rotating Element Pattern

Each book rotates three things while keeping two things constant:

| Rotates per book | Stays constant |
|-----------------|----------------|
| Backpacker (new audience surrogate) | Nate's first-person voice |
| Threat type (aquatic → botanical → marine) | The Clean-Up Crew as network |
| Setting / industry (cane → irrigation → reef) | Working-class pragmatism |

This prevents series fatigue while maintaining reader investment in the core engine.

## The Inheritance Pattern

Book N+1 must explicitly document:
1. What knowledge the crew carries forward (frequency monitoring, disruptor tech)
2. What scars they carry (physical, emotional, relational)
3. What new limitation forces innovation (old weapon doesn't work, old strategy fails)
4. What new character provides the missing skill (Lena = engineering; next book = marine biology)

## The Cost-of-Victory Rule

Each book must cost something permanent:
- Book 1: Bluey dies, Toby leaves, Nate/Maya relationship is fragile
- Book 2: Old Bill's health fails, farmers lose their crop, the crew becomes an institution
- Book 3+: TBD, but must escalate (character death, conversion, permanent separation)

Victory without cost makes the threat feel fake.
